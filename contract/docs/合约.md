- 我已经在本地起链并成功部署合约，可直接用于前端联调。

**如何启动本地链（终端重启后）**
- 启动节点：在 `contract` 目录运行 `npm run node`
- 端口：`http://127.0.0.1:8545/`（JSON-RPC）
- 链 ID：`31337`

**如何前端联调**
- 连接本地链（Wagmi/RainbowKit 自定义链配置示例）
```
export const localhost = {
  id: 31337,
  name: 'Hardhat Local',
  nativeCurrency: { name: 'ETH', symbol: 'ETH', decimals: 18 },
  rpcUrls: { default: { http: ['http://127.0.0.1:8545'] }, public: { http: ['http://127.0.0.1:8545'] } },
  blockExplorers: { default: { name: 'Local', url: 'http://localhost' } },
  testnet: true,
}
```
- 在前端配置中使用 `localhost` 链并把合约地址设为 `0x5FbDB2315678afecb367f032d93F642f64180aa3`，ABI 从 `artifacts/contracts/SimpleAuction.sol/SimpleAuction.json` 读取 `abi` 字段。
- 交互方法与 Monad 测试网一致：`getAuction`、`auctionCount`、`createAuction`、`placeBid`、`endAuction`，参考合约接口 `contract/contracts/SimpleAuction.sol:31-99,101-126`。

**发布到本地（合约代码变更后）**
- 编译：`npm run compile`
- 部署到本地：`npm run deploy:localhost`
- 部署结果会写入前端配置：`front/src/config/deployed.json`
  - 示例：
  ```
  {
    "contractAddress": "0x...",
    "chainId": 31337,
    "deployTime": "YYYY-MM-DDTHH:mm:ss.sssZ"
  }
  ```
  - 前端需确保连接 `localhost` 链（`front/src/config/wagmi` 中使用本地链配置）。

**切换到 Monad 测试网**
- 配置：
  - `MONAD_RPC_URL=你的Monad RPC`
  - `PRIVATE_KEY=你的部署私钥`
- 发布：`npm run deploy:monad`
- 网络配置位置：`contract/hardhat.config.js:14-20`

**测试**
- 运行合约单测：`npm test`
- 常见建议：
  - 变更合约后，优先跑测试再部署到本地
  - 若本地节点重启，链状态重置，需要重新 `deploy:localhost`

**说明**
- 部署脚本会自动写入前端地址文件：`front/src/config/deployed.json`
- 本地网络无需 `.env` 配置；测试网需提供 `MONAD_RPC_URL` 与 `PRIVATE_KEY`
